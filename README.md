# TypeScript Practice

## 利用手順

```bash
# パッケージをインストール
$ npm install

# TypeScript変更時の自動コンパイル
$ npm run watch

# Lint＋Format実行
$ npm run lint:fix

# Node実行
$ node xxxxx.js
```
